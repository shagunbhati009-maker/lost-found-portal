# Lost and Found Portal

This project is a Lost and Found Portal developed using PHP and MySQL.

## Features
- User Registration
- User Login
- Report Lost Item
- Report Found Item
- Search Items
- View Found Items
- Image Upload

## Technologies Used
- PHP
- MySQL
- HTML
- CSS
- Bootstrap
- XAMPP Server

## How to Run the Project
1. Install XAMPP
2. Move the project to htdocs folder
3. Import the database in phpMyAdmin
4. Open localhost/lostfound
